Primary ID is the column that contains the primary identifier for each row. This is the column that will be used to match with other data files or metadata that have the same primary identifier.
Fold Change is the column that contains the fold change values for each row. If this is not log2 transformed, check the box to perform the log2 transformation.
P-Value is the column that contains the significance for each row. If this is not -log10 transformed, check the box to perform the -log10 transformation.
